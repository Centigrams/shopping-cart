# Shopping Cart

## Overview

Sample e-commerce website with react-router

## Installation

```shell
git clone https://github.com/Centigrams/shopping-cart.git
cd shopping-cart
npm install
npm start
```

## Made with:

- [React](https://reactjs.org/)
- [Material UI](https://mui.com/)
- [React Router v6](https://reactrouter.com/)
